# How to set up initalization to github io

* CHECK <span style="color:brown">_config.yml</span> FILE
```
title: note blog.title
tagline: subtitle of title

description: >-
  describe your blog target

url: "https://username.github.io"

github:
  username: github name
  
social:
  name: social.name
  email: user@email.com
  links:
    - https://github.com/username

```

* REVISE GITHUB SETTING "PAGES"
    - Build and deployment source <span style="color:blue">"Deploy from a branch"</span> to <span style="color:blue">"Github Actions"</span>

YOU MAKE IT
